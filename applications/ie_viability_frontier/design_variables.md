# Design Variables — Viability Frontier Analysis

| Variable | Range | Default | Source |
|---|---|---|---|
| Sale price per unit | €160k-€480k | County median from PPR/HPM09 | CSO |
| Construction cost per sqm | €2,000-€3,500 | Regional from Buildcost H1 2025 | Buildcost.ie |
| Average unit size (sqm) | 70-140 | 110 | SCSI typical new home |
| Land cost per hectare | €100k-€1,600k | County median from RZLPA02 | CSO |
| Density (units/ha) | 20-100 | 40 | Planning norms |
| Development contributions | €5k-€30k | County-specific | LA schemes |
| Part V obligation (%) | 0-20 | 20 | Affordable Housing Act 2021 |
| Finance rate (annual) | 0.05-0.15 | 0.10 | Market range |
| Build duration (years) | 1.5-4.0 | 2.5 | Industry average |
| Developer profit margin (% GDV) | 0.10-0.25 | 0.175 | RICS guidance range |
